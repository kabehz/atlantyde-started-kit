## 🚀 Refrescar la lista

Para obtener los últimos contribuidores automáticamente:

```bash
git shortlog -sn --no-merges
```

O activa el plugin de MkDocs `git-authors-plugin` para integrar directamente el historial Git en cada página del sitio.
